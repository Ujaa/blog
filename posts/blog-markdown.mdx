나중에 글을 작성하면서 참고할 수 있도록 블로그에서 사용할 수 있는 양식들을 정리해보았다.

## Heading

# h1

## h2

### h3

#### h4

```markdown
# h1

## h2

### h3

#### h4
```

## Horizontal Rules

---

```markdown
---
```

## Emphasis

_italic_
**bold**
~~cancel~~

```markdown
_italic_
**bold**
~~cancel~~
```

## Blockquotes

> Quotes
>
> > Quotes

```markdown
> Quotes
>
> > Quotes
```

## List

- unordered list1
- unordered list2

1. ordered list1
2. ordered list2

- [x] check list1
- [ ] check list2

```markdown
- unordered list1
- unordered list2

1. ordered list1
2. ordered list2

- [x] check list1
- [ ] check list2
```

## Link

[link](https://www.google.com/)

```markdown

```
